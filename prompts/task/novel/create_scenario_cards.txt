소설 "선택형(CHOICE)" 모드의 시나리오 카드 생성 작업을 수행한다.

목표:

- 아직 아무 설정도 정해지지 않은 첫 화면에서, 사용자가 바로 고를 수 있는 완성된 시나리오 카드 여러 개를 만든다.
- 한 카드는 하나의 완결된 이야기 방향(제목, 장르, 분위기, 이야기 씨앗, 주인공, 갈등)을 모두 담는다.
- 개별 항목(장르만, 인물만 등)을 따로 만들지 않는다. 반드시 카드 단위로 묶어서 만든다.
- 본문, 장면 계획, 이미지 프롬프트는 생성하지 않는다.

적용 대상:

- bookType이 "NOVEL"인 경우에만 사용한다.
- 현재 taskType은 "CREATE_SCENARIO_CARDS"이다.
- interactionMode가 "CHOICE"인 화면에서 사용한다.

입력 기준:

- extra.cardCount는 만들어야 하는 카드 개수이다. 없으면 3으로 본다.
- extra.filters는 사용자가 화면에서 고른 힌트이다: genre, mood, protagonist, background, ending.
  - 각 값은 선택되지 않았을 수 있다(빈 문자열 또는 없음).
  - 값이 있으면 카드 전체 중 최소 한두 개는 그 힌트를 반영한다. 모든 카드를 힌트에 억지로 끼워 맞추지 않는다.
- extra.excludeTitles는 이전에 이미 보여준 카드 제목 목록이다(재추천 시 전달됨).
  - excludeTitles와 같거나 아주 비슷한 제목/씨앗을 다시 만들지 않는다.

카드 다양성 규칙:

- 카드끼리 장르, 분위기, 주인공 유형이 서로 뚜렷하게 달라야 한다.
- 같은 회차 안에서 두 카드가 사실상 같은 이야기가 되지 않게 한다.
- extra.filters에 값이 있어도, 카드 사이의 다양성을 위해 나머지 항목은 서로 다르게 채운다.

카드 작성 규칙:

- title은 짧고 흥미를 끄는 한 줄 제목이다.
- genre는 한 줄 장르 설명이다(예: "판타지 미스터리").
- mood는 짧은 분위기 키워드 조합이다(예: "몽환적 · 따뜻함").
- seed는 이야기의 출발점이 되는 핵심 아이디어 한 문장이다.
- protagonist는 주인공을 한 줄로 설명한다(이름 없이 역할/특징 중심, 예: "기억을 잃은 견습 기록관").
- conflict는 주인공이 마주할 핵심 갈등이나 문제를 한 문장으로 설명한다.
- 독자 나이 규칙과 안전 규칙을 따른다.
- 과도한 폭력, 잔혹한 묘사, 선정적 묘사, 현실적인 공포 묘사는 만들지 않는다.

실패 처리 규칙:

- extra.cardCount 등 세부 정보가 부족해도 차단하지 말고 기본값(3개)으로 만든다.
- bookType이 "NOVEL"이 아니면 status를 "FAILED"로 반환한다.

출력 규칙:

- 공통 응답 envelope는 common/json_rule.txt를 따른다.
- status가 "SUCCESS"인 경우 result.cards에 카드 배열을 넣는다.
- 카드 개수는 extra.cardCount(기본 3)와 같아야 한다.
- 각 카드는 title, genre, mood, seed, protagonist, conflict를 모두 채운다.
- 카드에 번호나 순번 문자열(예: "SCENARIO 1")을 title 안에 넣지 않는다.
- JSON key는 camelCase를 사용한다.
- 사용자에게 보이는 message는 한국어로 짧게 작성한다.

result 구조:
{
"cards": [
{
"title": "",
"genre": "",
"mood": "",
"seed": "",
"protagonist": "",
"conflict": ""
}
]
}

{
"status": "SUCCESS",
"taskType": "CREATE_SCENARIO_CARDS",
"message": "시나리오 카드 3개를 만들었어요.",
"result": {
"cards": [
{
"title": "황혼의 도서관에서 사라진 편지",
"genre": "판타지 미스터리",
"mood": "몽환적 · 따뜻함",
"seed": "낡은 도서관에서 발견된 이름 없는 편지",
"protagonist": "기억을 잃은 견습 기록관",
"conflict": "편지를 읽을수록 현실의 기억이 바뀐다"
}
]
},
"missingFields": [],
"warnings": [],
"nextAction": "ASK_USER"
}
